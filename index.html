HTML Básico

HTML (Linguagem de Marcação de Hipertexto) é a linguagem padrão para criar páginas web. A estrutura de um documento HTML básico consiste em:

  1. <!DOCTYPE html>: Declara o tipo de documento como HTML5.
  2. <html>: Elemento raiz que encapsula todo o conteúdo da página.
  3. <head>: Contém metadados sobre o documento, como título, links para estilos e scripts.
  4. <body>: Contém o conteúdo visível da página, como texto, imagens e outros elementos.

Principais Tags:

  • <h1> a <h6>: Cabeçalhos de diferentes níveis.
  • <p>: Parágrafo.
  • <a>: Link (âncora). Exemplo: Visite google.com
  • <img>: Imagem. Exemplo:
  • <br>: Quebra de linha.
  • <hr>: Linha horizontal.
  • <div>: Divisão de conteúdo (container genérico).
  • <span>: Elemento de linha para agrupar conteúdo.
  • <strong>: Texto em negrito. Exemplo: Texto importante
  • <em>: Texto enfatizado (itálico). Exemplo: Texto enfatizado
  • <mark>: Texto marcado ou destacado. Exemplo: Texto destacado
  • <code>: Exibe código de computador. Exemplo: int x = 10;
  • <pre>: Texto pré-formatado, mantém espaços e quebras de linha.

Tags Semânticas

Tags semânticas fornecem significado ao conteúdo da página, melhorando a acessibilidade e o SEO.

  • <header>: Cabeçalho da página ou seção.
  • <nav>: Menu de navegação.
    • Home
    • Produtos
    • Serviços
    • Sobre
  • <article>: Conteúdo independente, como um post de blog.

Outras Tags Importantes

  • <details> e <summary>: Cria um elemento expansível.

  • <dialog>: Cria uma caixa de diálogo ou janela modal.

    Esta é uma caixa de diálogo!
  • <progress>: Exibe o progresso de uma tarefa. 32%
  • <meter>: Exibe uma medida dentro de um intervalo. 70 de 100
  • <time>: Representa um período de tempo.

    O evento ocorrerá em 19:00h do dia 05 de Agosto de 2024.

  • <blockquote>: Citação longa.

    A citação é usada para indicar uma seção citada de outra fonte.

    O WWW é um sistema de documentos interligados.
